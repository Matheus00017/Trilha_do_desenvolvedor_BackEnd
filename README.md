# Controle de estoque


Sistema web para o controle de estoque de produtos e categorias, desenvolvido em **ASP.NET Core MVC** com operações completas de CRUD e persistência em **SQL Server**.

## Tecnologias utilizadas

- ASP.NET Core MVC (.NET 10)
- Entity Framework Core
- SQL Server LocalDB
- Bootstrap 5
- C#

- ## Funcionalidades

- ✅ Cadastro, edição, visualização e exclusão de **Categorias**
- ✅ Cadastro, edição, visualização e exclusão de **Produtos**
- ✅ Relacionamento entre Produto e Categoria
- ✅ Barra de pesquisa por nome ou categoria
- ✅ Interface em português

- ## Como rodar o projeto

### 1. Clonar o repositório
git clone https://github.com/Matheus00017/Trilha_do_desenvolvedor_BackEnd.git

### 2. Abrir o projeto
- Abra a pasta ControleDeEstoque
- Abra o arquivo ControleDeEstoque.sln no Visual Studio

### 3. Criar o banco de dados
Abra o Console do Gerenciador de Pacotes e execute:
Update-Database

### 4. Executar o projeto
Pressione F5 no Visual Studio
